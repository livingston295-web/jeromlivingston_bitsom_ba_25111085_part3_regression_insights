# Regression-Based Business Insights & Model Interpretation

## Business Problem Summary

The objective of this project is to identify factors associated with monthly sales using regression analysis. The analysis supports leadership in making decisions related to marketing, inventory, staffing, and store performance.

## Dataset Description

The dataset contains retail store information including:

* Monthly Sales
* Marketing Spend
* Footfall
* Average Discount Percentage
* Staff Count
* Inventory Availability
* Customer Rating
* Region
* Store Type

## Dependent Variable

* Monthly Sales

## Independent Variables

* Marketing Spend
* Footfall
* Inventory Availability
* Customer Rating
* Average Discount Percentage
* Store Type (Dummy Variable)

## Regression Approach

The project includes:

* Two Simple Linear Regression Models
* One Multiple Regression Model
* Residual Analysis
* Model Comparison

## Dummy Variable Approach

Store Type was converted into dummy variables.

Reference Category:

* High Street

Dummy Variable Used:

* Mall Dummy

## Model Comparison Summary

The multiple regression model provides better explanatory power than the individual simple regression models because it considers several business variables together.

## Final Model Selected

Multiple Regression

## Business Recommendation

Increase investment in effective marketing, improve inventory availability, and monitor customer footfall to improve monthly sales performance.

## Assumptions

* Linear relationships exist between variables.
* Data is accurate.
* Errors are randomly distributed.

## Limitations

* Regression measures association, not causation.
* Other business variables may influence sales.
* Historical data may not fully predict future performance.
